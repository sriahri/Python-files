Python is an interpreted language and the python interpreter is a computer program thatexecutes code written in the python programming language. The python interactive interpreteris useful forsimple operations or programs consisting of only a few lines. For more lines, a programmercan writepython code in a file and then provide that file to the interpreter.For normal development a program installs python as an application on a local computer.Macintosh and Linux operating systems usually include python, while windows does not.Programmers can download that latest version of python for free.
